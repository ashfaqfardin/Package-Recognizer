# Data

## Overview
This directory contains datasets used for model training and testing. All data files are stored in extracted format, ready for machine learning pipelines.

## Accessing Data
Extracted datasets are available in the following Google Drive repository:
https://drive.google.com/drive/folders/1qeTjO90zWc45lGwgF7DqgLb-mmiFoLj-?usp=sharing

## Usage
Download the extracted data files from the Google Drive link and place them in this directory before running data preparation, training, or inference workflows.

## Data Preparation
For information on data cleaning and preprocessing procedures, refer to the [training_and_data_cleaning.ipynb](../notebooks/training_and_data_cleaning.ipynb) notebook.
