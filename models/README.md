# Models

## Overview
This Google Drive directory contains trained machine learning models exported in serialized format for inference and deployment purposes.

## Accessing Trained Models
Pre-trained models are available in the following Google Drive repository:
https://drive.google.com/drive/folders/1OxZww1lAyfiMygizYQcjqISgzzx_Ydl2?usp=sharing

## Model Files
- **Format**: All exported models are stored in pickle (`.pkl`) format
- **Deployed Model**: `pkg_recognizer_v1.pkl` - The primary model used for production inference

## Usage
Download the required model files from the Google Drive link and place them in this directory before running inference or deployment pipelines.
